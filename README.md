## 🎯 Architecture

<div align="center">

```mermaid
flowchart LR
    A[📄 Documents] -->|Chunking| B[🧩 Text Chunks]
    B --> C[🧠 Embeddings]
    C --> D[(⚡ FAISS)]
    E[👤 User Query] --> F[🔍 Retriever]
    F --> D
    D --> G[📚 Context]
    G --> H[🤖 FLAN-T5]
    H --> I[✨ Answer]
```
